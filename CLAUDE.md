# Local AI Meetings

Local-first meeting transcription and AI summarization toolkit.

## Project Structure

- `Makefile` — main entry point for processing recordings
- `meetings/` — output directory for transcripts and summaries (gitignored)
- `.agents/skills/` — AI skill definitions (meeting-summary, follow-up-email)

## Workflow

1. Drop a video file (`.mov`, `.mp4`, `.webm`, `.mkv`, `.avi`) in the project root
2. Run `make process P=Name1_Name2` to extract audio and transcribe
3. Output goes to `meetings/YYYY_MM_DD_Name1_Name2/`
4. Use skills to summarize or draft follow-up emails from the transcript

## Dependencies

- **ffmpeg** — audio extraction from video
- **whisper** (openai-whisper) — speech-to-text transcription

## Conventions

- Meeting directories: `meetings/YYYY_MM_DD_Participants/`
- Transcripts: `.txt` files generated by whisper
- Summaries: `.md` files in the same directory
- Follow-up emails: `.email.md` files in the same directory

## Skills

Skills are defined in `.agents/skills/`. Read the relevant `SKILL.md` before executing a skill.
